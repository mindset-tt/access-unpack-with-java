# Repo layout

`access-unpack-core/`
- Cross-platform extraction library

`access-unpack-cli/`
- Picocli-based command-line interface

`access-unpack-ui/`
- Swing desktop UI for cross-platform usage

`windows-helper/`
- PowerShell automation contract for Microsoft Access

`scripts/`
- Local build and run helpers

`docs/`
- Architecture, layout, and implementation notes

`generated-runs/`
- Recommended location for extracted artifacts and UI-created output trees
- Intended to stay out of git so source and recovered artifacts remain separated
